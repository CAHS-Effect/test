Neon + Netlify Setup

1 Go to app.neon.tech, create a project, copy the Connection string (starts with postgresql://).
2 In Netlify → Site → Environment Variables, add:
   DATABASE_URL = your Neon connection string
   CMS_SECRET = any strong password you choose (this protects write access)
3 Enter that same CMS_SECRET value below and click Save & Test.
4 Redeploy your Netlify site (or trigger a new deploy) after adding the env vars.

Note: The secret is stored only in this browser's localStorage. Tables are created automatically on first connection.
